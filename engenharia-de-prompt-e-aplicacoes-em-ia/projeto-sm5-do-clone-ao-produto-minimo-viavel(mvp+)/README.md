# Projeto SM5 - Do Clone ao Produto Mínimo Viável (MVP+)
 
## 📝 Descrição do Projeto
Neste projeto, conduzi a evolução de um clone funcional para um MVP+ com diferenciais claros de produto e justificativas técnicas de priorização.
 
## 🚀 Tecnologia Utilizada
AI Studio.

## 📊 Resultados e Aprendizados
O projeto priorizou diferenciais estratégicos como histórico local, pin de arquivos, busca por modal e estatísticas em tempo real para otimizar a experiência de edição. A decisão técnica focou em recursos de alto impacto no fluxo de trabalho, garantindo a entrega de valor máximo dentro de um escopo controlado. No aspecto analítico, o desenvolvimento do MVP+ exigiu um equilíbrio rigoroso entre tempo de implementação e o custo cognitivo da interface. O aprendizado central foi validar que a relevância prática de cada função define a aceitação do usuário final. Como resultado, a plataforma entrega uma ferramenta robusta que prioriza a produtividade sem comprometer a simplicidade.
