# Projeto SM2 - Laboratório de Classificação Visual
 
## 📝 Descrição do Projeto
Desenvolvimento de um modelo de classificação visual utilizando o Google Teachable Machine. O projeto explora na prática o ciclo completo de um modelo de aprendizado supervisionado (treinamento, validação e inferência).
 
## 🚀 Tecnologias Utilizadas
Teachable Machine 
 
## 📊 Resultados e Aprendizados
Durante a utilização da plataforma, executei o pipeline completo de IA (coleta, treino, validação e testes). Para garantir a confiabilidade do classificador, tomei a decisão técnica de balancear as amostras de cada classe, reduzindo o viés de predição. O principal aprendizado empírico deste laboratório foi comprovar como pequenas variações no dataset afetam drasticamente a estabilidade e a precisão geral das previsões.
 
## 🔧 Como Executar

### Passos
1. Acesse o laboratório: <https://teachablemachine.withgoogle.com/models/dOSbmmhTs/>
2. Teste o modelo com imagens de entrada.
3. Compare as probabilidades de saída por classe.
