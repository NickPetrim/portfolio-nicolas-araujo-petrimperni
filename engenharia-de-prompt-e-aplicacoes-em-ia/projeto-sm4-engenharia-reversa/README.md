# Projeto SM4 - Engenharia Reversa
## 📝 Descrição do Projeto
Nesse projeto, realizei a engenharia reversa de uma plataforma de geração de QR Codes para desconstruir sua arquitetura e mapear decisões críticas de interface e usabilidade. O objetivo foi converter essas observações em requisitos técnicos acionáveis, garantindo que a nova implementação evitasse a cópia literal e priorizasse a diferenciação funcional estratégica.

## 🚀 Tecnologias Utilizadas
TypeScript e AI Studio

## 📊 Resultados e Aprendizados
O projeto consolidou o desenvolvimento de uma plataforma avançada de QR Codes, integrando React 19 e IA generativa para automatizar a criação de conteúdos complexos via linguagem natural. A implementação do algoritmo de Scannability Intelligence permitiu validar a funcionalidade do design em tempo real, garantindo alta taxa de leitura. Aprendi a estruturar interfaces Dark Mode responsivas utilizando Tailwind CSS e componentes modulares, priorizando a privacidade com processamento local no navegador. A experiência refinou o domínio sobre bibliotecas de renderização vetorial (SVG) e a manipulação de estados complexos para personalização extrema. Como resultado, o UniCode superou geradores convencionais ao unir ferramentas de design profissional com análise técnica de dados.
