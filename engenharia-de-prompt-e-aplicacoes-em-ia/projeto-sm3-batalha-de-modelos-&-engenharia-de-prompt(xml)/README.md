# Projeto SM3 - Batalha de Modelos e Engenharia de Prompt (XML)
 
## 📝 Descrição do Projeto
Neste experimento, avaliamos a performance de diferentes modelos de IA ao serem submetidos ao mesmo desafio técnico. Para garantir a imparcialidade e o controle total do teste, utilizamos uma estrutura de Prompt Engineering baseada em XML, segmentando claramente contexto e diretrizes. O objetivo foi validar qual modelo apresentava a melhor consistência semântica e completude técnica, respeitando rigorosamente as restrições de formato estabelecidas.

## 🚀 Tecnologias Utilizadas
* **Linguagem:** XML
* **Ferramentas:** Modelos de IA
 
## 📊 Resultados e Aprendizados
O projeto alcançou resultados sólidos em ambiente de teste, demonstrando a eficácia da estruturação de prompts em XML para o controle de modelos de IA. O modelo Claude atingiu a performance ideal, cumprindo exatamente a proposta com excelência semântica e sem apresentar erros de sintaxe. Através da análise comparativa, aprendeu-se que a verbosidade (quantidade de tokens) está diretamente ligada à excelência do resultado, onde modelos com maior geração de dados entregaram maior detalhamento técnico. Além disso, a implementação destacou o DeepSeek pela criatividade visual e uso de efeitos dinâmicos, enquanto o Claude foi consolidado como a ferramenta de escolha para prototipagem rápida e códigos complexos devido à sua superioridade na compreensão da estrutura solicitada.
